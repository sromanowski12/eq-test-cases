## Selenium Installation

1. Check your version of python
```py
python --version
```

2. Download and install pip
```py
curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
python get-pip.py
```

3. Install selenium
```py
pip install -U selenium
```
